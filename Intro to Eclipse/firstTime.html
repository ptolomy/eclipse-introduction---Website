First Time Setup
In this section we will explain what happens during the first time you setup Eclipse.
If you followed the previous video on Install an IDE then you have successfully installed Eclipse to your system.
Either double click the Icon on your desktop for Eclipse or search in the start menu for it. You will see the launch frame on the desktop (0.12) which will then be replaced by the option of creating your workspace.
On the left side you will see a 4 options,
Launch which will take you to the development screen, Workspaces, which allows you to manage your workspaces.
Recent which shows you any recent work you have opened.
and how to which is a helpful tool to get you started yourself and is a fantastic feature for those already familiar with these stages of setting up an IDE.

At (0.59) you can see that once you have selected a workspace you then launch using that workspace using the Arrow next to it or simply click on an already created workspace.
