Creating a Workspace
A workspace is simply a location on a storage medium where you store any projects you are working on. in the video you can see how to create a new workspace and some that are already created.

When you select a new workspace you can select a preferred location or use the suggested default location and name it as you see fit.

In the video example you can see there are already some workspaces created with super creative names for them as well. I'm particularly proud of Eclipse Workspace2 but in this case each one refers to different project sources.
For example newEWorkspace is where University projects are kept and Eclipse workspace is personal projects I work on in my own free time and you can create as many as you need to keep things separate though it may be a good idea to use a better naming convention for your own workspaces.
