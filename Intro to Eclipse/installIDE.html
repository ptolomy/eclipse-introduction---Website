Installing an IDE
In this video you can see where to go to download (0.00- 0.07) this takes you to the next page where you select the Eclipse version you want the primary version of Eclipse is the one we are covering here.(0.12)
The next page shows you where you are getting the download from(Server location) and the sha key used to generate the download so you can check it against the version with the installation so you know it is correct and hasn't been modified.(0.15)
Once you have selected the download your browser will alert you to this and give you choices such as Run or Save and Cancel, in this demo I select run (0.24).
This opens up the Eclipse download manager(0.41).
There are a lot of choices here for the various environments available but for this we will look at the Java Developer IDE.
Upon choosing your desired IDE, you are presented with an installation screen letting you choose which Java Virtual Machine to use (1.8) in this case and where you would like to install the software you can also choose to include a start menu and desktop shortcut.(0.46)
Once you begin installing you are given a progress bar and you can cancel the install at any time during this (0.57).
